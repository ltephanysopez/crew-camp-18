Some of the Awesome Stuff You'll Learn

HTML5 Markup

HTML, or HyperText Markup Language, is primarily used in the creation of the webpage skeleton. All other languages incorporated in web development are implemented around HTML, and are spaced in specific areas according to the tags used in the HTML code.

CSS Styling

CSS, or Cascading Style Sheets, is the simplest of the three languages. While HTML is used to create the practical structure of a website, CSS is used to style the elements of the webpage that are created through HTML. Colors, fonts, borders, and other characteristics are all controlled through CSS.

JavaScript

JavaScript, an object-oriented programming language, is used to create interactive progarms such as games, apps, and other interfaces. It is commonly used in conjunction with HTML to create websites that interact with their users.
